# A very basic code on how to center an element on the page.

###### In HTML we have a `<button>`
  ```
  <body>
    <button>Hover Me</button>
  </body>
  ```
###### In CSS the centering
  ```
body {
    display: flex;
    align-items: center;
    justify-content:center;
    height: 100vh;
}
  ```
